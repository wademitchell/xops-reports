TL;DR
Andrew Warner and guest Peter Cooper (editor of JavaScript Weekly) walk through the most-starred open source AI repos of the week. The hot themes are agent plumbing (context, scraping, design taste, file conversion) and free local alternatives to big-name products. The standout: Riley Brown's two-prompt Lovable clone built with Claude's new "Fable" model.
Creator's Key Callouts
  • lfnovo/open-notebook is the standout NotebookLM alternative because it runs locally and supports custom voice profiles (not stuck with the same two NPR hosts)
  • mvanhorn/last30days-skill is the #1 repo of the week; scrapes X/YouTube/Reddit using your browser cookies if you don't have API keys
  • rbrown101010/rilable is a Lovable clone built in roughly two prompts with Claude's new "Fable" model; Andrew wouldn't recommend replacing Lovable yet
  • Leonxlnx/taste-skill fixes the "every AI site looks the same" problem, but Peter warns you're trading Claude's default cliches for another set of cliches
  • apple/container is macOS's WSL moment: native Linux containers on Apple Silicon with file system sync
  • microsoft/markitdown can drop token usage by up to 70% by converting PDFs/Office/YouTube into clean markdown before sending to Claude
  • The "Phantom Stars" tool flagged suspicious star engagement on several of the most popular repos
  • Andrew had a self-deprecating moment admitting he got in his own head trying to be a "real PM" for the pm-skills segment
