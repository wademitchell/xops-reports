TL;DR
Simon shows how to build specialist AI advisors in Notion by combining three parts: agent instructions, reusable skills, and a curated knowledge base sourced from books or trusted frameworks. The main pitch is that grounded agents give better, less generic advice, and can be refined with ingestion workflows plus a self-improving knowledge loop.

Creator's Key Callouts
  • The core build has 3 parts: instructions, skills, and a dedicated knowledge base.
  • The knowledge base is the key differentiator because it grounds answers in curated sources instead of generic model behavior.
  • He recommends using specialist agents in chat via Notion's personal agent, not team custom agents, for this consulting-style workflow.
  • He uses a Meta Agent to help generate the sub-agent instructions, skill pages, and knowledge base structure faster.
  • He advises reviewing AI-written instructions and skills manually for clarity, boundaries, and token efficiency.
  • He warns not to overload the context window by constantly interrupting or stuffing too much into the prompt.
  • He suggests confidence fields, categories, and linked source material so the agent can prioritize proven ideas over unvalidated ones.
