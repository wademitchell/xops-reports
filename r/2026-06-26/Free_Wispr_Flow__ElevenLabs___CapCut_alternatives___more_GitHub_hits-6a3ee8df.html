TL;DR
Andrew Warner and Adam Brakhane run through the week's buzziest AI and open source GitHub projects, with a strong focus on agent-friendly video editing, local voice tools, coding-agent memory, forecasting, and browser or web access for agents. The big theme is control: self-hosted or local tools that replace paid SaaS apps while giving AI agents more direct, flexible workflows.
Creator's Key Callouts
  • OpenMontage is the standout: it lets a coding agent research, script, edit, and render a polished video from a short prompt, using real footage by default rather than AI-looking generated clips.
  • codebase-memory-mcp matters because it gives coding agents codebase awareness and memory while cutting wasted token usage on large repos.
  • FluidVoice impressed Adam enough that he tried it after recording and said he was canceling his other dictation subscription.
  • OpenCut is promising but not yet a great editing experience; Andrew thinks agent-native editing workflows may be a better direction than simply cloning CapCut.
  • Andrew asks viewers to help him get Matt Van Horn on the show and to like, subscribe, or leave critical feedback so the reports keep improving.
